# CESI Bot Widget

JS-виджет чат-бота для работы с Flowise API.

## Настройка Git

Для подключения к GitHub выполните следующие команды:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Dhmnzr79/cesi-agent.git
git push -u origin main
```

Если возникнут проблемы с блокировкой файлов, закройте Cursor и другие программы, использующие git, и попробуйте снова.
